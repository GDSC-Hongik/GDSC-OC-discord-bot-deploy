# GDSC 홍익 오픈 커뮤니티 배포

## 세팅

1. [도커](https://docker.com) 설치

2. `.env` 파일 생성

   - [GDSC-OC/discord-bot](https://github.com/GDSC-OC/discord-bot) 참조

3. `serviceAccountKey.json` 파일 생성

   - [GDSC-OC/discord-bot](https://github.com/GDSC-OC/discord-bot) 참조

4. 실행

   ```
   docker compose up --build
   ```
